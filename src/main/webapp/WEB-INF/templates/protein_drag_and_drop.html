Some RNA gets transformed into protein, this is usually done by something called a ribosome. This ribosome takes three nucleotides from the RNA, this is called a codon. This codon binds an amino acid. A protein slowly gets created like a string of beads, where the beads are the amino acids. This string folds into a proper functioning protein, but we're getting ahead of ourselves.


For now, we're doing what the ribosome should be doing. We're translating the codons to their respective amino acid. We can code a program to do this, but it is good to get a hang on what is happening before we let a program run it for us. A codon table will help us. The first ring is the first nucleotide, the second ring represents the second nucleotide, and the third ring represents the third nucleotide. To read it, just go from in to out of the circle.
